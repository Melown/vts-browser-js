# Melown API

## Documentation
* [Melown API](https://github.com/Melown/melown-js/wiki/Melown-API)

## Install

```
npm install
```

(`yarn install` will be ready soon)

## Build

```
webpack
```

## Run dev server

```
webpack-dev-server --content-base build/ --inline
```

And go to http://localhost:8080/demos/core/basic/
